Лабораторна робота #3: Класи та об’єкти. Конструктори та деструктори. Модифікатори. Інкапсуляція


Завдання 1. 
- Створити клас з атрибутами та конструктором. У методі main() ініціалізувати створення екземплярів класу та продемонструвати роботу його методів згідно умов завдання.
- Скласти опис класу для визначення одновимірних масивів строк фіксованої довжини. Передбачити контроль виходу за межі масиву, можливість звернення до окремих рядків масиву за індексами, виконання операцій поелементного зчеплення двох масивів з утворенням нового масиву, злиття двох масивів з виключенням повторюваних елементів, а також виведення на екран елемента масиву по заданому індексу та виведення всього масиву. Зробити властивості класу приватними, а для їх читання створити методи-геттери.

Завдання 2
- Створити у попередньому завданні два методи з використанням серіалізації та десеріалізації JSON: 
Метод 1. Зберігає створений об’єкт класу з Завдання 1 у JSON файл 
Метод 2. Відкриває JSON файл з даними та створює об’єкт класу з цими даними для виконання Завдання 1.
